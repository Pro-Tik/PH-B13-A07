## KeenKeeper

**KeenKeeper** is a personal friendship management application crafted to act as your digital shelf of meaningful connections. It empowers you to seamlessly browse, tend to, and proactively nurture the relationships that matter most to you before they go cold.

### 🛠️ Technologies Used
- **Next.js (App Router)** - For blazing-fast server-side rendering and elegant routing mechanics.
- **React** - Built entirely with functional components and interactive hooks.
- **Tailwind CSS & daisyUI** - For creating a flawless, beautiful, and fully responsive user interface natively.
- **Recharts** - Used to render complex, animated, data-driven SVG pie charts effortlessly.
- **React-Toastify** - To provide satisfying, real-time alert notifications upon user interaction.

### ✨ Key Features

1. **Global Dynamic Timeline Logging**
   Interact with your friends organically via fully functional action buttons (Calls, Texts, Meetups, Snooze, Archive, or Delete). Every action you log is instantly dispatched across a global application state and documented historically within a searchable filterable timeline feed.

2. **Interactive Visual Analytics Dashboard**
   Say goodbye to guesswork. An intuitive, highly modern pie chart dynamically aggregates exactly how you are maintaining your personal relationships—compiling raw numbers into interactive, hovering tooltip data to visualize whether you are heavily relying on phone calls, texting, or genuine meetups over time.

3. **Intelligent Responsive Design**
   A seamless user-experience constructed for tiny screens, tablets, and enormous ultrawide desktop monitors alike. Incorporates advanced CSS grid layouts, intelligent responsive breakpoints, and custom hamburger-dropdown navbars to ensure nothing ever looks scattered or squished.
